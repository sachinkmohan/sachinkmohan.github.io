Here are some Git commands that I frequently use in my projects:

1. Remove git tracking

```bash
git rm -r --cached .astro/

// rm = remove from git
// -r = recursive
// --cached = only remove from git index, keep local files
// .astro/ = directory to remove from git tracking
```
